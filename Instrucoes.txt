COMO RODAR O PROJETO
====================

1. Gerar os dados de teste:
   cd geradorBasePratica1
   javac *.java
   java GeradorArquivosCommitsTeste
   cd ..

2. Compilar o projeto:
   javac -d bin src/modelo/*.java src/estruturas/*.java src/ordenacao/*/*.java src/util/*.java src/Main.java

3. Executar:
   java -cp bin Main

O programa vai processar os 3 arquivos JSON (1k, 10k, 100k commits) e gerar um relatorio.txt com os resultados.
